## ルーティング

- :railway_track: `/front/index`
  - Home 画面

- :file_folder: apps
  - :memo: `index.py`
    - Home 画面の裏側の処理を行う
- :file_folder: html
  - :memo: `index.html`
    - Home 画面の html 部分
- :file_folder: static
  - :memo: `favicon.png`
    - favicon

  - :memo: `styles.css`
    - 独自の css スタイルを使いたい時に利用する
